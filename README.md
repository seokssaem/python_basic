## 🚀 사용 방법

### 로컬에서 실행하기

1. 프로젝트 폴더에서 `index.html` 파일을 더블클릭하거나
2. 브라우저에서 `index.html` 파일을 열어주세요

### 웹 서버로 실행하기 (선택사항)

Python이 설치되어 있다면:

```bash
# Python 3
python -m http.server 8000

# 브라우저에서 http://localhost:8000 접속
```

또는 VS Code의 Live Server 확장을 사용할 수 있습니다.

4. **저작권 표시**: 모든 페이지 하단에 저작권 정보 명시

## 🔄 콘텐츠 추가 방법

새로운 주제의 이미지를 추가하려면:

1. `images/` 폴더에 새 폴더 생성
2. 해당 폴더에 이미지 파일 추가 (순서대로 번호 부여)
3. 해당 주제 HTML 파일에서 이미지 경로 업데이트

예시:
```html
<div class="image-section">
    <img src="../images/폴더명/이미지파일명.jpg" 
         alt="설명" 
         class="content-image">
</div>
```
